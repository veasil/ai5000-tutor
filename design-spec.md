# 设计规范 (Design Spec) — Editorial / 描边气泡风（v2）

> 此规范定义全站统一视觉语言。**v2 起全面改为「奶油底 + 黑描边 + 硬投影 + editorial 衬线」的杂志/轻 neo-brutalist 风格**（参考 variant 共享设计的对话气泡英雄区）。
> 产品语境：6-15 岁儿童的闯关式 AI 创客学习产品；气质 = 友好、清晰、有手作质感的轻游戏感。

## 0. 最高原则
1. **气泡即对话**：核心容器是「带黑描边的大圆角气泡/卡片」，小伍说的话用气泡承载，整页是小伍一步步引导的对话流。
2. **描边 + 硬投影定形**：用 **2px ink 描边 + 硬投影（offset 向下、blur 0、ink 色）** 给容器体积感，**不再用柔和模糊阴影**（英雄气泡可加粗到 3px / offset 10）。
3. **强调色稀缺**：橙 `$primary` 只用于**强调**（名字高亮、当前态、小贴纸点缀）；**主操作按钮改 ink 黑胶囊**，不是橙。蓝色作中性/未解锁。

## 1. 颜色（角色制）
- **底**：`$bg-cream` #FAF5EC（页面）/ `$surface` #FFF（气泡卡）/ `$surface-warm` #FFFBF4。
- **墨**：`$ink` #2C2017 —— 描边、硬投影、正文、**黑胶囊主按钮**底。`$ink-soft` #6E6052 次文 / `$ink-mute` #A89A89 标签。
- **强调橙**：`$primary` #F5871F —— 名字/当前关高亮、状态字、贴纸；`$primary-deep` #B85C09 橙底深字。
- **中性蓝**：`$blue*` 仅作未解锁/信息态。
- **语义**：`$danger` #C0492E / `$danger-soft` #FBEAE5（安全红灯/错误）。
- 文字对比过 WCAG AA（正文 4.5:1，大字 3:1）。

## 2. 深度系统（描边 + 硬投影）
- **卡片 / 气泡**：`fill $surface` + `stroke $ink` 2px + 硬投影 `{offset:{x:0,y:4}, blur:0, color:$ink}` + 大圆角（卡 18 / 气泡 30）。
- **英雄对话气泡**：`stroke $ink` 3px + 硬投影 `{y:10, blur:0}` + 圆角 30。
- **像素母题**（关卡块 / 火花格）：实色填充 + 2px 描边 + 小硬投影，保留直角/小圆角游戏感。
- 分隔用 1px `$line` 细线或留白；不要柔和模糊阴影。

## 3. 间距 (8pt Grid)
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64。组件内边距 16-24（气泡 36-40）；区块间距 24-32；页面边距 [40,72]。同组紧、跨组松。

## 4. 字体与排版（Editorial）
- **标题 + 正文：`$font-serif` = Noto Serif SC**（`$font-display` 也指向它）—— 杂志衬线质感。
- **名字 / 关键词强调**：橙色 `$primary` + 斜体（如英雄区「小伍。」）。
- **小标签 / 数值 / LV**：`$font-body` Noto Sans SC（字重 600、letterSpacing 1-2）或 `$font-pixel` Press Start 2P，制造与衬线的 mono 对比（仿参考的 spec 档案格）。
- **小伍口吻短句**：可用 `$font-fun` ZCOOL KuaiLe 作温暖点缀（非必须）。
- 字号阶梯：英雄名 56-62 · H1 30-40 · H2 22 · H3 18 · Body 16-17 · Caption 13 · Label 11 · Pixel 9。
- 行高：标题 1.0-1.2，正文 1.6-1.7；正文每行 50-75 字符。

## 5. 形状
- 圆角：气泡/英雄 30 · 卡片 18 · chip/小元件 8-12 · 按钮 pill(两头圆)。
- **主按钮 = ink 黑胶囊**（白字）；次按钮 = 描边白胶囊；安全相关 = `$danger` 胶囊。
- 像素直角母题（关卡路径/火花格/徽章）保留，描边 + 硬投影区分态。

## 6. 状态（必画）
每个数据界面都要有：**空 / 加载骨架 / 出错 / 安全红灯**。语义色见 §1。文案主动语态、不用「Oops!」「！」。

## 7. 可访问性
WCAG AA 对比；可点击目标 ≥ 40px；状态不只靠颜色（配 ✓/🔒/▶ 图标或文字）。

## 变更记录
- **v2**：全面转向 editorial 描边气泡风。英雄区 = 左侧大大的小伍立绘（透明悬浮，奶油底）+ 右侧黑描边对话气泡（衬线大标题 + 自我介绍 + ink 黑胶囊按钮 + 分隔线 + 2×2 档案格），行动项之外的卡片在第二屏向下展开。`$font-display`→Noto Serif SC；主按钮橙→ink；柔和阴影→硬投影+描边。
